# Garage Antananarivo Showcase

Site vitrine premium pour un garage automobile a Antananarivo.

Objectif : inspirer confiance, expliquer les services, puis convertir vers une demande de rendez-vous ou un message WhatsApp.

## Stack

- Next.js
- React
- TypeScript
- Vercel-ready

## Commandes

```bash
npm install
npm run dev
npm run build
```
